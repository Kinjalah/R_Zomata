🍽️ Zomato Customer Preference & Pricing Analytics using R
📌 Overview
This project performs end-to-end statistical data analysis on 7,105+ restaurant records from the Zomato dataset using R.
The objective is to uncover customer preferences, pricing trends, cuisine demand, restaurant density, and rating behavior using statistical methods and predictive modeling.

🎯 Objectives
Identify most common restaurant types
Analyze top cuisines by popularity
Find most expensive and budget-friendly restaurants
Study online ordering and table booking trends
Explore restaurant concentration by area
Examine relationship between ratings and average cost
Predict ratings using linear and multiple regression
🛠️ Tech Stack
Language: R
Libraries: dplyr, tidyr, ggplot2
Techniques:
Data Cleaning
Missing Value Imputation
Descriptive Statistics
Data Visualization
Linear Regression
Multiple Regression
Predictive Analysis
📊 Key Insights
Quick Bites and Casual Dining are the most common restaurant types
North Indian and Chinese cuisines dominate demand
Higher-priced restaurants show slightly better ratings
Online ordering is widely adopted
Cost + number of ratings significantly impact predicted ratings
📈 Models Used

Predicts customer ratings based on average cost.
